Comparação dos algoritmos

Algoritmos Utilizados

  1. Bubble Sort: Algoritmo simples que compara cada par de elementos adjacentes e os troca se estiverem na ordem errada. Complexidade temporal: (O(n^2)).
  2. Selection Sort: Algoritmo que encontra o menor elemento e o coloca na posição correta, repetindo o processo para os elementos restantes. Complexidade temporal: (O(n^2)).
  3. Insertion Sort: Algoritmo que constrói a lista ordenada inserindo um elemento de cada vez na posição correta. Complexidade temporal: (O(n^2)) no pior caso, (O(n)) no melhor caso.
  4. Merge Sort: Algoritmo de divisão e conquista que divide a lista em sub-listas, ordena cada sub-lista e as combina. Complexidade temporal: (O(n \log n)).

Análise dos Resultados

Os resultados mostram que o Merge Sort é mais eficiente para grandes tamanhos de entrada, enquanto Bubble Sort, Selection Sort e Insertion Sort apresentam desempenho inferior, especialmente em grandes conjuntos de dados.

Algoritmos de Ordenação

Bem-vindo ao guia sobre Algoritmos de ordenação! Este documento tem como objetivo apresentar alguns algoritmos e um pouco de suas propiedades.

Introdução aos Algoritmos

Bubble Sort

O Bubble Sort é um algoritmo simples de ordenação. Ele percorre a lista várias vezes, comparando elementos adjacentes e trocando-os se estiverem na ordem errada. Esse processo é repetido até que a lista esteja completamente ordenada.

Selection Sort

O Selection Sort também é um algoritmo de ordenação simples. Ele divide a lista em duas partes: uma parte ordenada e outra não ordenada. A cada iteração, ele encontra o menor elemento da parte não ordenada e o coloca no final da parte ordenada.

Insertion Sort

O Insertion Sort é outro algoritmo simples de ordenação. Ele percorre a lista da esquerda para a direita, inserindo cada elemento na posição correta da parte ordenada da lista.

Merge Sort

O Merge Sort é um algoritmo de ordenação baseado em dividir para conquistar. Ele divide a lista em duas metades, ordena cada metade separadamente e então combina as duas metades ordenadas para obter a lista final ordenada.
